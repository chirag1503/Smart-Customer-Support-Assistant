# 💬 Smart Customer Support Assistant

An **AI-powered full-stack web application** that automates customer query responses using **Flask**, **REST APIs**, and a lightweight **NLP intent classification model** trained with Scikit-learn (94 % accuracy).  
Built to simulate real-world customer support systems where users can chat in real time and receive instant, intelligent answers.

---

## 🚀 Features

- 🧠 **AI-Driven Responses** – Detects user intent (e.g., card issue, refund, account help) using TF-IDF + Logistic Regression.
- 🔗 **Flask REST APIs** – Backend handles message routing, AI prediction, and analytics.
- 💬 **Interactive Chat UI** – Simple, responsive interface built with HTML, CSS, and JavaScript.
- 📊 **Analytics Ready** – `/analytics` endpoint provides intent frequency data for dashboards.
- ⚙️ **Data-Driven Design** – Built to simulate enterprise-grade customer service workflows.
